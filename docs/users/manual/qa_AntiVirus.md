# Anti-Viren Service
<!-- icon: icon_virus.svg -->

Um den Antivirus-Service zu nutzen, muss ClamAV (hierbei handelt es sich um einen Standard-Open-Source-Virenscanner) in tine konfiguriert werden.

Sofern das geschehen ist, werden alle Dateien (ausgenommen E-Mail-Anhänge) die in tine zur Verfügung stehen, wöchentlich auf Viren, Trojaner, Malware und andere Gefahren geprüft. Sollte der Scan positiv sein, wird diese Datei entsprechend markiert.
