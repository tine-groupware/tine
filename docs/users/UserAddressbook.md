Problemlösungen im Adressbuch-Modul von tine

HowTo: Suchen von allen Kontakten, die mit _keinem_ Lead verknüpft sind
=================

"Verknüpft mit" ... "definiert durch" ... "Lead"

-> so erhält man alle gesuchten Kontakte
